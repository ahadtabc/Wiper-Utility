System Requirements & Setup

Bluering is built on a robust Python framework. Please ensure the following prerequisites are met for a smooth installation and operation.

1. Python Environment

Bluering requires **Python 3.x** (latest stable version recommended).

  • Download: https://www.python.org/downloads/
  • IMPORTANT: During installation, you MUST check the **"Add Python to PATH"** box.

2. Required Python Packages

These modules must be installed via **pip** using an Administrative Command Prompt:

pip install customtkinter psutil wmi speedtest-cli pystray pillow pywin32

Note: If running from a provided executable, these steps may be optional, but are necessary for running the source code (`Main_Code REG SIMU.py`).

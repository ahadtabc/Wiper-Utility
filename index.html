Application Architecture

Wiper's core challenge is running a cleanup task during system shutdown. This is achieved through a clever integration with the Windows Task Scheduler, which listens for a specific system event. The diagram below illustrates this event-driven workflow.

1. User Initiates Shutdown/Restart

The process starts when the user clicks Shutdown or Restart in Windows.

‚Üì

2. Windows Logs Event ID: 1074

Windows logs this event to the System event log, signaling a shutdown process.

‚Üì

3. Task Scheduler Trigger

A pre-configured task detects Event 1074 and launches `wiper_trigger.exe`.

‚Üì

4. Wiper Popup Appears

The trigger script displays a modal popup, temporarily pausing the shutdown.

‚Üì

5a. User clicks "Clear & Shutdown"

The script calls the core cleaning logic and then re-issues the system shutdown command.

5b. User clicks "Cancel"

The script issues a shutdown abort command, and the system returns to normal.

Core Cleaning Logic

Deployment & Packaging

Once the Python scripts are complete, they must be packaged into executable files (`.exe`) and configured to run at shutdown. This involves using PyInstaller and setting up a task in the Windows Task Scheduler.

Step 1: Package with PyInstaller

Run these commands in your terminal to create single-file, windowless executables for both the main app and the shutdown trigger. Make sure you have a `.ico` file for your logo.

pip install pyinstaller
pyinstaller --onefile --windowed --icon=new.ico wiper_core.pypyinstaller --onefile --windowed --icon=new.ico wiper_trigger.py

Step 2: Configure Task Scheduler

Create a new task that triggers on the shutdown event.

  • Open Task Scheduler and select "Create Task...".
  • Go to the Triggers tab and click "New...".
  • Set "Begin the task:" to On an event.
  • Configure the settings: Log: System, Source: User32, Event ID: 1074.
  • Go to the Actions tab, click "New...", and set the action to "Start a program".
  • Browse to and select your compiled `wiper_trigger.exe`.
  • Save the task. Wiper is now active.
